# Roblox Account Generator
```
Expect bugs ;-)

Roblox account generator built in PYTHON;
Works by using the selenium lib and the random-username lib

Captchas are MANUAL

When registered an account successfully it will save the following information: USER:PASS:COOKIE
In a file called users.txt
It will then log-out and make a new account
```

# Requirements
```
pip install seleniun
pip install random_username
pip install time
pip install os
```
